-This program is a chess game written in C. To win the game the player needs to connect a specific number of chess horizontally, vertically or in diagonal.

-The main.c program demostrates three different games with chess board of size 7 * 6 and requirement of connecting 4 chess.

-To see the game, please run ./game in the VS Code terminal.

-This program is one of the past assignment projects. The purpose of it is to demonstrate the coder's ability in programming with C.
